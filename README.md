### Demos

mivideo: 用 TypeScript 编写，仿造小米官网底部视频播放的组件，由弹出框组件和视频播放组件组合而成。

#### 使用：

clone 项目

1. 切换目录 `cd mivideo`

2. 下载依赖 `npm install`

3. 启动项目 `npm start`
